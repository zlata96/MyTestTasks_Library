## 2. Online Shoping App 

App for online shoping with ability to sign in and log in, also user can search goods in main screen and interact with profile. Some information is gotten from [API](https://run.mocky.io/v3/a9ceeb6e-416d-4352-bde6-2203416576ac). I used [Moya](https://github.com/Moya/Moya) for network layer. Test task was completed with [SnapKit](https://github.com/SnapKit/SnapKit) for layout with code and  [Realm](https://github.com/realm/realm-swift) for saving data

More details and code watch [here](https://github.com/zlata96/onlineShop_TestTask)
